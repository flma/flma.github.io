java环境搭建

  • java

#什么是java？
java是一种面向对象程序设计（oop）语言，在java中万事万物皆对象；java是一种跨平台的开发语言，实现了“一次编写，随处运行”。

#如何安装java环境？

  • 下载jdk1.8 点击下载，因为我是64位系统，所以选择下载x64的版本
  • 点击安装包，出现安装界面
  • 点击下一步，更改安装路径，这里我安装到C:/java/jdk1.8目录下。
  • 点击下一步，会弹出对话框，提示安装jre，更改安装路径C:/java/jre1.8
  • 点击下一步，出现下面界面，点击关闭
  • 配置环境变量，计算机→属性→高级系统设置→高级→环境变量
  • 新建系统变量JAVA_HOME，填写idk的安装路径c:/java/jdk1.8
  • 系统变量→寻找 Path 变量→编辑。在变量值最后输入 %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
    （注意原来Path的变量值末尾有没有;号，如果没有，先输入；号再输入上面的代码)
  • 系统变量→新建CLASSPATH 变量,变量值填写 .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（注意最前面有一点）
